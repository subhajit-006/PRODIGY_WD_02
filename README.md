# ⏱️ Professional Stopwatch Web Application

A lightweight, responsive, and highly accurate stopwatch web application. This project features a clean user interface and essential time-tracking functionalities, including lap timing.

## ✨ Features

  * **Start/Pause/Resume:** Smooth control over the timer state.
  * **Lap Tracking:** Record and display specific time intervals without stopping the main clock.
  * **Reset Functionality:** Clear all data and timer values with a single click.
  * **Precise Timing:** Displays hours, minutes, seconds, and milliseconds.
  * **Responsive UI:** Fully functional on mobile, tablet, and desktop screens.

## 🛠 Tech Stack

  * **HTML5:** Structured the timer display and control buttons.
  * **CSS3:** Applied modern styling, centering, and button hover effects.
  * **JavaScript (Vanilla):** Implemented `setInterval()`, logic for time formatting, and lap management.

## 🧠 Core Functionality

The application uses a custom logic to format the time string:

1.  **Time Logic:** Calculates elapsed time by tracking total milliseconds.
2.  **Formatting:** Ensures leading zeros (e.g., `01:05:09`) for a professional digital clock look.
3.  **DOM Manipulation:** Dynamically appends lap times to a list for user review.
